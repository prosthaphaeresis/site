## Vectors

Vectors are stored contiguously in memory and support native arithmetic operators (`+`, `-`, `*`, `/`) for immediate evaluations.

```lua title="vectors.lua"
-- 2D Vector creation and linear operations
local velocity = math.vector2.new(10.5, 20.5)
local acceleration = math.vector2.new(2.0, 4.0)
local force = velocity + acceleration // [!code highlight]
local impulse = force * 3.0

-- 3D Vector spatial displacement and normalization
local position = math.vector3.new(5.0, 10.0, 15.0)
local destination = math.vector3.new(1.0, 2.0, 3.0)
local displacement = position - destination // [!code highlight]
local direction = displacement:normalized() // [!code highlight]

-- 4D Vector coordinates for projection weights
local coordinate = math.vector4.new(1.0, 2.0, 3.0, 4.0)
local weight = math.vector4.new(2.0, 2.0, 2.0, 2.0)
local projection = coordinate + weight

```

## Quaternions & Matrices

For spatial tracking, coordinate projections, and complex directional rotations without gimbal lock, use the built-in matrix and orientation types.

```lua title="matrices.lua"
-- Initialize baseline identity transformations
local transform = math.matrix4.identity() // [!code highlight]
local tensor = math.matrix3.identity()

-- Quaternion rotation sequencing
local rotation = math.quaternion.new(0.0, 0.0, 0.0, 1.0)
local orientation = math.quaternion.new(0.7071, 0.0, 0.0, 0.7071)

-- Combine spatial orientations using cross-multiplication
local spin = rotation * orientation // [!code highlight]
local inverted = spin:inverse() // [!code highlight]

```

## Transformations

Apply structural changes to entities by binding linear algebra elements directly to data blueprints within your running instance state.

```lua title="transformations.lua"
local world = world.new()

-- Register layout bindings with fallback defaults
local position = world:component("position", math.vector3.new(0, 0, 0))
local transform = world:component("transform", math.matrix4.identity()) // [!code highlight]

-- Instantiate an active tracking registry index
local actor = world:spawn()

-- Attach spatial math data arrays to the tracking context
world:attach(actor, position, math.vector3.new(0, 10, 5)) // [!code highlight]
world:attach(actor, transform, math.matrix4.identity())

```
