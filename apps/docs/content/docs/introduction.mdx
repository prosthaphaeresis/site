Prostha, derived from the word prosthaphaeresis, is a lightweight, open-source simulation engine written in C++ for any device, such as IoT devices, microcontrollers, or mobile devices/desktops.
It handles complex graphical computations using deterministic, low-level memory management for speed, but exposes clean Lua bindings for high-level scripting—giving you the performance of C++ without the headache of manual pointer tracking.
It’s cross-platform, highly efficient, and easy to embed anywhere.
